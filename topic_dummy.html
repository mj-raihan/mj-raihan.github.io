    The Autistic Kids Companion leverages advanced machine learning and computer vision technologies to provide an accurate and effective exercise assessment for autistic children and individuals with motor dysfunction. One of the core technologies powering the app is Google MediaPipe, a state-of-the-art framework for real-time pose tracking. By analyzing key body landmarks, the app can determine movement accuracy, suggest corrections, and provide meaningful feedback to users.


    To ensure scalability and efficiency, the app utilizes cloud computing platforms like Amazon AWS and Google Cloud for model training, data processing, and storage. These cloud-based solutions enable the app to handle large volumes of data efficiently, making it easier to improve its predictive capabilities over time. Additionally, the KIMORE Benchmark Dataset, a well-known dataset for movement analysis, is incorporated into the app’s learning process. Using SHAP-based assessment (SHapley Additive exPlanations), the app provides interpretable machine learning predictions, ensuring that the exercise recommendations are tailored to the specific needs of each user.
